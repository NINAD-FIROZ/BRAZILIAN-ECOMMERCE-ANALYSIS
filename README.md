# 🛒 Olist Brazilian E-Commerce — Exploratory Data Analysis

An end-to-end data analysis project using real e-commerce data from Olist, a Brazilian marketplace platform with 100,000 orders from 2016–2018.

---

## 📌 Business Questions Answered

1. Which states generate the most revenue?
2. How satisfied are customers overall?
3. What percentage of orders are successfully delivered?
4. How long does delivery take on average?
5. Does delivery time directly affect customer satisfaction?

---

## 🔍 Key Findings

- **São Paulo dominates revenue** — generating 5.2M BRL, nearly 3x second-place Rio de Janeiro
- **75% of customers are satisfied** — rating their orders 4 or 5 stars
- **97%+ of orders are delivered successfully** — fulfilment operations are reliable
- **Average delivery time is 12 days** — with a long tail of 30+ day deliveries
- **Delivery speed drives satisfaction** — 1-star customers waited 19 days on average vs 10 days for 5-star customers

---

## 🛠️ Tools Used

- Python 3
- pandas — data loading, merging, cleaning, analysis
- matplotlib — data visualization

---

## 📂 Dataset

[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) — available on Kaggle

The dataset consists of 9 relational CSV files. This project merges 4 of them:
- `olist_orders_dataset.csv`
- `olist_customers_dataset.csv`
- `olist_order_items_dataset.csv`
- `olist_order_reviews_dataset.csv`

---

## 🚀 How to Run

1. Clone this repository
2. Download the dataset from Kaggle and place the CSV files in the same folder as the notebook
3. Open `olist_project.ipynb` in Jupyter
4. Run **Kernel → Restart & Run All**
